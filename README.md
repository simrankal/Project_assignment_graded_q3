# CodeJudge SQL Query Implementation

## Objective

This project implements SQL queries on the normalized CodeJudge database schema.

The queries demonstrate:

* filtering
* joins
* aggregation
* HAVING clauses
* subqueries
* set logic

Each query is validated using actual dataset outputs.

## Files

| File             | Purpose                        |
| ---------------- | ------------------------------ |
| queries.sql      | All SQL queries                |
| query_outputs.md | Sample outputs and validations |
| sql_reasoning.md | SQL concept explanations       |

## Execution

Run queries after importing:

* schema.sql
* dataset CSVs

Example:

```sql id="56sjcs"
SOURCE queries.sql;
```
